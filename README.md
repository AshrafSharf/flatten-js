# Javascript library for 2d geometry

## Installation

  `npm install flatten-js`

## Usage

    var Flatten = require('flatten-js');

    var point = new Flatten.Point(1,1);
      
## Tests

  `npm test`

## Documentation

You can see documentation here https://alexbol99.github.io/flatten-js/index.html
